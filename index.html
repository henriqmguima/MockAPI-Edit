SPAWT

Acesse, insira, edite e delete os conteúdos de uma API de filmes

Spawt é um site de conteúdos audiovisuais, utilizando do serviço MockaApi onde você pode acessar ao nome, diretor, data de lançamento e poster de filmes. Para conferir as informações inclusas no site e poder adicionar novos filmes acesse o site TMDB.
